# Lead Generation System - Quick Start

## Start the Backend API

```bash
cd "c:\Users\rajni\OneDrive\Desktop\Ai 1\back-end"
python api.py
```

The API will start on: http://localhost:8000

## Open the Frontend Dashboard

Simply open this file in your browser:
```
c:\Users\rajni\OneDrive\Desktop\Ai 1\frontend\index.html
```

Or use a local server:
```bash
cd "c:\Users\rajni\OneDrive\Desktop\Ai 1\frontend"
python -m http.server 3000
```

Then visit: http://localhost:3000

## Features

✅ View all leads in a beautiful table
✅ Filter by name, company, email, status, and score
✅ Add new leads manually
✅ Edit existing leads
✅ Delete leads
✅ Run the AI agent from the UI
✅ Export leads to CSV
✅ Real-time stats dashboard
